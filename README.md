# Interactive Wall Calendar 🗓️

Welcome to your **interactive wall calendar**! This project brings a visually stunning, fully functional calendar to the browser, complete with date range selection, integrated notes, dynamic visuals, and extra features for a seamless user experience.

---

## 🌸 Calendar Aesthetic

- **Dynamic hero images:** Change with the month (sunflowers for April, snow for December, cherry blossoms for March, etc.).  
- **Elegant typography:** Playfair Display serif font for month names, giving a premium physical calendar feel.  
- **Realistic details:** Decorative binding holes, warm parchment color palette, natural shadows, and subtle textures.  
- **Immersive design:** Month-specific accent colors that shift the overall UI palette.

---

## 📅 Date Range Selector

- **Interactive selection:** Click to set start and end dates.  
- **Live hover preview:** Highlights dates dynamically as you hover.  
- **Clear visual states:** Circles for today, filled dots for start/end, and tinted range fill between.  
- **Day count:** Automatically shows the number of days in your selected range.

---

## 📝 Integrated Notes

- **Persistent memo pad:** Month-level notes with lined paper texture and handwriting font.  
- **Range-specific notes:** Add notes and optionally pin them to selected date ranges.  
- **Color-coded:** Notes are visually coded by date and filtered according to your selection.  
- **Local storage:** All notes saved in the browser, so nothing is lost on refresh.

---

## ✨ Additional Features

- **Dark mode toggle:** Smooth themed transition for day/night viewing.  
- **Holiday markers:** US holidays highlighted with a legend below the calendar.  
- **Dot indicators:** Dates with notes show a visual marker.  
- **“Today” button:** Quickly jump to the current month.  
- **Month-specific imagery:** UI palette changes subtly for each month to enhance aesthetics.
